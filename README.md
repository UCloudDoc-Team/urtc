# 概览


实时音视频（UCloud Real-Time Communication，简称URTC），依托于UCloud强大的覆盖能力，可以实现用户的就近接入，提供网络低延迟、低丢包率的音视频通信。

URTC SDK支持Web、Android、IOS、MAC OS、Windows、Electron多种客户端的集成接入。通过集成URTC SDK，能快速构建多人音视频通话、一对多或多对多的实时互动直播。
